Step-1: Team Gathering, Collaboration and Select the Problem Statement
Problem Statement:

How can we build a free, accessible, and feature-rich video conferencing platform — "Smart Meet" — that enables individuals and teams to connect remotely through real-time audio/video communication, screen sharing, in-meeting chat, session recording, and flexible meeting scheduling, without requiring expensive subscriptions or third-party dependency for core functionality?

Context & Background:

With the steady rise of remote work, online education, and global collaboration, the demand for reliable video conferencing tools has grown significantly. Dominant platforms (Zoom, Google Meet, Teams) impose feature restrictions on free tiers and raise privacy concerns. There is a clear opportunity to build a lightweight, open-source-friendly application that provides essential conferencing features at zero cost to the end user, using modern web technologies such as React, Node.js, Socket.io, and the Agora RTC SDK.
